    • kinematics: displacement
      Displacement
      In physics, displacement essentially means the "change in position". The equation for displacement is: $$\triangle x=x_f-x_i$$ Displacement is a vector quantity, which means it can be expressed with direction.
    • kinematics: velocity
      Velocity
      Velocity is the rate of change. It is a vector quantity, which means it can be expressed with direction.
      Average velocity
      Average velocity tells you the total displacement over total time, defined as: $$\bar{v}=\frac{\triangle x}{\triangle t} = \frac{x_f-x_i}{t_f-t_i}$$
      Instantaneous velocity
      Instantaneous velocity is the velocity of an object at an exact point. To find the instantaneous velocity, look at the graph of a position time graph and calculate the slope. It is defined as: $$v=\lim_{x \rightarrow 0} \frac{\triangle x}{\triangle t}$$ If we are given the equation of the graph, we can take the first derivative of the function and it will give us the instantaneous velocity.
    • kinematics: acceleration
      Acceleration
      Acceleration is the rate of change of velocity. It is a vector quantity, which means it can be expressed with direction.
      Average acceleration
      Average velocity tells you the total velocity over total time, defined as: $$\bar{a}=\frac{\triangle v}{\triangle t} = \frac{v_f-v_i}{t_f-t_i}$$
      Instantaneous acceleration
      Instantaneous acceleration is the acceleration of an object at an exact point. To find the instantaneous acceleration, look at the graph of a position time graph and calculate the slope. It is defined as: $$v=\lim_{t \rightarrow 0} \frac{\triangle v}{\triangle t}$$ If we are given the equation of the graph, we can take the first derivative of the function and it will give us the acceleration.
    • kinematics: 1-d motion
      1-D Motion
      We can calculate problems using many different kinematic equations based on which knows are given to us. All equations are based on velocity, acceleration, and time.
      1. $$\bar{v}=\frac{\triangle \overrightarrow{x}}{t}$$
      2. $$\bar{a}=\frac{\triangle \overrightarrow{v}}{t}$$
      3. $$v_f=v_i+at$$
      4. $$\triangle \overrightarrow{x}=\overrightarrow{v}_it+\frac{1}{2}at^2$$
      5. $$v_f^2=v_i^2+2a \triangle x$$
      6. $$\triangle x=t(\frac{v_i+v_f}{2})$$
      7. $$v=at$$
      8. $$\triangle \overrightarrow{x}=\overrightarrow{x}_f-\overrightarrow{x}_i$$
    • kinematics: motion graphs
      Motion Graphs
      We can express displacement, velocity, acceleration, and time in terms of graphs. Often, the terms for these graphs are displacement-time, velocity-time, and acceleration-time graphs.
      Displacement-time graph
      A displacement-time graph represents the position of an object at a certain time.

      From the graph, using the cartesian plane, we can deduce:
      1. 1. The average velocity and instantaneous velocity of the object from taking the slope. If the slope is positive, velocity is positive and if negative, velocity is negative. However if the $slope=0$, there is no movement.
      Velocity-time graph
      A velocity-time graph represents the velocity of an object at a certain time.

      From the graph, using the cartesian plane, we can deduce:
      1. 1. The average acceleration and instantaneous acceleration of the object are found from taking the slope. If the slope is positive, acceleration is positive and if acceleration, acceleration is negative. However if the $slope=0$, the acceleration is constant.
      2. 2. To find the distance from the velocity-time graph, find the area under the slope. If you are given the function, integrate it to find the distance.
      3. 3. If the slope crosses under the $x$ axis, then the object switches direction.
      Acceleration-time graph
      A acceleration-time graph represents the acceleration of an object at a certain time.

      From the graph, using the cartesian plane, we can deduce:
      1. 1. If the $slope=0$, the velocity is constant.
      2. 2. To find the velocity from the acceleration-time graph, find the area under the slope. If you are given the function, integrate it to find the velocity.
      3. 3. If the slope crosses under the $x$ axis, then the object switches direction.

      Free fall
      Free fall describes an object that is influenced by the force of gravity. The magnitude is expressed in terms of $g$: $$g \approx 10m/s^2$$
