    • prerequisites: functions
      Finding the domain
      The domain of the function tells you where the function for values of $x$ are defined. It can be expressed in either interval notation or as an inequality.

      In general:
      1. 1. If the function is a linear, polynomial, or exponential, then all $x$ values exist.
      2. 2. If the function is a rational, then set the denominator equal to zero. The value given means that $x$ values exist everywhere except at the point(s).
      3. 3. If the function is a square root, then ony $x$ greater than or equal to zero exist.
      4. 4. If the function is a logarithm, then only postive $x$ values exist.
      Finding the range
      The range of the function tells you where the function for values of $y$ are defined. Simiar to the domain, it can be expressed in either interval notation or as an inequality. Usually, graphing the function is the best way to find the domain.

      Algebraically, you can find the domain of the inverse of the function; that is the function's range.
      Finding x-intercepts (zeroes)
      The x-intercepts tell you when a function intersects the $x$-axis. You solve $x$-intercepts by making $f(x)=0$.

      1. 1. If it is rational, you make the numerator of $f(x)=0$
      2. 2. If it is unfactorable, use the quadratic formula: $\frac{-b\pm\sqrt{b^2-4ac}}{2a}$
      3. 3. If it is greater than a quadratic (ex. cubic) and unfactorable, use synthetic division.
      Finding y-intercepts
      The y-intercepts tell you when a function intersects the y-axis. You solve y-intercepts by making $f(x)=0$.
      Finding even-odd symmetry
      To find whether the function is even or odd, plug in $-x$ for $x$. If the signs given are:

      1. 1. The same as the original function, then the function's symmetry is even.
      2. 2. Opposite from the original function (plus signs become negative and vice versa), then it is odd.
      3. 3. Something different, then it is neither even or odd.

      If the function's symmetry is said to be even, then that means the function is symmetrical when reflected over the y-axis. Contrarily, if the functiion is said to be odd, then the function is the same as rotates over the y-axis 180 degrees.
      Finding inverse functions
      To find the function's inverse, replace all $y$ with $x$ and $x$ with $y$, then solve for $y$. A function's inverse is denoted by a superscript -1: $f^{-1}(x)$.
    • prerequisites: asymptotes
      Asymptotes
      An asymptote is an imaginary line in which the function approaches at a certain point but never touches it. There are three types, vertical, horizontal, and oblique asymptotes.
      Finding vertical asymptotes
      To find a vertical asymptote, merely set the denominator of the function equal to zero.
      Finding horizontal & oblique asymptotes
      To find horizontal asymptotes, look at the function's powers.
      Let's say we have the function $\large \frac{ax^y}{bx^z}$.
      1. 1. If $y < z$ then the horizontal asymptote exists at $y=0$.
      2. 2. If $y = z$ then the horizontal asymptote exists at $\large y=\frac{a}{b}$ where $a$ and $b$ are the coefficients of the numerator and denominator.
      3. 3. If $y > z$ then there are no horizontal asymptotes; however, there an oblique asymptote exists. You find it by using long division.

      Trigonometric Identities
      Triangle Identities:
      1. $sin^2\theta+cos^2\theta=1$
      2. $1+tan^2\theta=sec^2\theta$
      3. $1+cot^2\theta=csc^2\theta$

      Sum and Difference Identities:
      1. $sin(a\pm b)=2sin\theta cos\theta$
      2. $cos(a\pm b)=cos^2\theta-sin^2\theta$
      3. $tan(a\pm b)=cos^2\theta-1$

      Double Angle Identities:
      1. $sin(2\theta)=2sin\theta cos\theta$
      2. $cos(2\theta)=cos^2\theta-sin^2\theta=cos^2\theta-1=1-sin^2\theta$
      3. $tan(2\theta)=\frac{2tan\theta}{1-tan^2\theta}$

      Half Angle Identities:
      1. $sin(\frac{\theta}{2}~)=\large \sqrt{\frac{1-cos\theta}{2}}$
      2. $cos(\frac{\theta}{2}~)=\large \sqrt{\frac{1+cos\theta}{2}}$
      3. $tan(\frac{\theta}{2}~)=\large \frac{1-cos\theta}{sin\theta}=\frac{sin\theta}{1+cos\theta}$

      Hyperbolic Trigonometric Identities
      Hyperbolic Trig Identities:
      1. $sinhx=\large \frac{e^x-e^{-x}}{2}$
      2. $coshx=\large \frac{e^x-e^{-x}}{2}$

      Triangle Identities:
      1. $cosh^2\theta-sinh^2\theta=1$
      2. $tanh^2\theta+sech^2\theta=1$
      3. $coth^2\theta+csch^2\theta=1$

      Derivatives involving trigonometry
      We can find the derivative of trigonometric functions which can be found through the limit definition of a derivative. Here they are:
      $$y=sinx \\ y'=cosx$$
      $$y=cosx \\ y'=-sinx$$
      $$y=tanx \\ y'=sec^2x$$
      $$y=cscx \\ y'=-cscxcotx$$
      $$y=secx \\ y'=secxtanx$$
      $$y=cotx \\ y'=-csc^2x$$
      Derivatives involving inverse trigonometry
      We can also find the derivative of the inverse trigonometric functions through the same methods as the regular trig. Here is a compilation:
      $$y=sin^{-1}x \\ y'=\frac{1}{\sqrt{1-x^2}}$$
      $$y=cos^{-1}x \\ y'=-\frac{1}{\sqrt{1-x^2}}$$
      $$y=tan^{-1}x \\ y'=\frac{1}{1+x^2}$$
      $$y=csc^{-1}x \\ y'=-\frac{1}{|x|\sqrt{x^2-1}}$$
      $$y=sec^{-1}x \\ y'=\frac{1}{|x|\sqrt{x^2-1}}$$
      $$y=cot^{-1}x \\ y'=-\frac{1}{1+x^2}$$
      Derivatives involving hyperbolic trigonometry
      Finally, here is a list of the derivatives involving hyperbolic trigonometry.
      $$y=sinhx \\ y'=coshx$$
      $$y=coshx \\ y'=sinhx$$
      $$y=tanhx \\ y'=sech^2x$$
      $$y=cschx \\ y'=-cschxcothx$$
      $$y=sechx \\ y'=-sechxtanhx$$
      $$y=cothx \\ y'=-csc^2x$$
      Integrals involving trigonometry
      $$\int sinx \ dx \\ =-cosx+c$$
      $$\int cosx \ dx \\ =sinx+c$$
      $$\int tanx \ dx \\ =-ln|cosx|+c$$
      $$\int cscx \ dx \\ =ln|cscx-cotx|+c$$
      $$\int secx \ dx \\ =-ln|secx-tanx|+c$$
      $$\int cotx \ dx \\ =ln|sinx|+c$$
      Integrals involving inverse trigonometry
      $$\int \frac{1}{\sqrt{1-x^2}} \ dx \\ =sin^{-1}x+c$$
      $$\int \frac{1}{1+x^2} \ dx \\ =tan^{-1}x+c$$
      $$\int \frac{1}{x\sqrt{x^2-1}} \ dx \\ =sec^{-1}|x|+c$$
