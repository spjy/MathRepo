    • derivatives: rules
      Derivatives
      A derivative finds the slope of a function. The notation to indicate a derivative can be expressed through either: $$f'(x)\ \text{or}\ \frac{d}{dx}(f(x))$$
      Limit definition
      You can find the derivative of a function through the limit definition of a derivative which is: $$\lim_{h \rightarrow 0}\frac{f(x+h)-f(x)}{h}$$
      APPLY. Find the derivative of the function using the limit definition of a derivative: $$f(x)=x^2$$ $$\lim_{h \rightarrow 0}\frac{(x+h)^2-x^2}{h}$$ $$\lim_{h \rightarrow 0}\frac{x^2+2hx+h^2-x^2}{h}$$ $$\lim_{h \rightarrow 0}\frac{2hx+h^2}{h}$$ $$\lim_{h \rightarrow 0}\frac{h(2x+h)}{h}$$ $$\lim_{h \rightarrow 0}2x+h$$ $$f'(x)=2x$$
      Power rule
      We are able to find the derivative of a function by simply applying the power rule. If we want to find the derivative of $f(x)=x^n$ we use the formula: $$nx^{n-1}$$ We bring down the exponent to the front and subtract one of the exponent.
      APPLY. Find the derivative of the function using the power rule. $$f(x)=x^6$$ $$f'(x)=6x^5$$
      Product rule
      Not only can we use the power rule, but we can also use the product rule if we have two components to the function. If we wanted to find the derivative of $f(x) \times g(x)$, the general formula is: $$f(x)g'(x)+g(x)f'(x)$$
      APPLY. Find the derivative of the function using the product rule. $$f(x)=(x^2)(x+1)$$ $$f'(x)=(x^2)(x+1)'+(x+1)(x^2)'$$ $$f'(x)=(x^2)(1)+(x+1)(2x)$$ $$f'(x)=(x^2)+2x^2+2x$$ $$f'(x)=3x^2+2x$$
      Quotient rule
      The third rule we can use is the quotient rule, and if we want to express it in terms of $\frac{f(x)}{g(x)}$ then it would be expressed by: $$\frac{g(x)f'(x)-f(x)g'(x)}{g^2(x)}$$
      APPLY. Find the derivative of the function using the quotient rule. $$f(x)=\frac{x+1}{x}$$ $$f'(x)=\frac{(x)(x+1)'-(x+1)(x)'}{x^2}$$ $$f'(x)=\frac{(x)(1)-(x+1)(1)}{x^2}$$ $$f'(x)=\frac{x-x-1}{x^2}$$ $$f'(x)=-\frac{1}{x^2}$$
      Chain rule
      The final rule we can use is the chain rule to deal with compositional functions. If we wanted to find the derivative of $(f(g(x))$, we'd use the following: $$f'(g(x)) \times g'(x)$$
      APPLY. Find the derivative of the function using the chain rule. $$f(x)=(x^2+1)^2$$ $$f'(x)=2(x^2+1)(2x)$$ $$f'(x)=(4x^2+2)(2x)$$ $$f'(x)=8x^3+4x$$
      Implicit differentiation
      We can also derive functions which have more than one variable, such as $x+y=1$. If we wanted to derive it in terms of $x$, first we would normally take the derivative of both sides. Then for each $y$ we derive we would simply multiply by a $y'$ (like applying the chain rule).
      APPLY. Find the derivative of the function in terms of $x$ using implicit differentiation. $$2x+y=3$$ $$(2x+y)'=(3)'$$ $$2+y \times y'=0$$ $$yy'=-2$$ $$y'=\frac{-2}{y}$$ $$y'=\frac{-2}{3-2x}$$
    • derivatives: trig
      Derivatives involving trigonometry
      We can find the derivative of trigonometric functions which can be found through the limit definition of a derivative. Here they are:
      $$y=sinx \\ y'=cosx$$
      $$y=cosx \\ y'=-sinx$$
      $$y=tanx \\ y'=sec^2x$$
      $$y=cscx \\ y'=-cscxcotx$$
      $$y=secx \\ y'=secxtanx$$
      $$y=cotx \\ y'=-csc^2x$$
      APPLY. Find the derivative of the function. $$f(x)=sin^2(2x)$$ $$f'(x)=2cos(2x) \times 2$$ $$f'(x)=4cos(2x)$$
      Derivatives involving inverse trigonometry
      We can also find the derivative of the inverse trigonometric functions through the same methods as the regular trig. Here is a compilation:
      $$y=sin^{-1}x \\ y'=\frac{1}{\sqrt{1-x^2}}$$
      $$y=cos^{-1}x \\ y'=-\frac{1}{\sqrt{1-x^2}}$$
      $$y=tan^{-1}x \\ y'=\frac{1}{1+x^2}$$
      $$y=csc^{-1}x \\ y'=-\frac{1}{|x|\sqrt{x^2-1}}$$
      $$y=sec^{-1}x \\ y'=\frac{1}{|x|\sqrt{x^2-1}}$$
      $$y=cot^{-1}x \\ y'=-\frac{1}{1+x^2}$$
      APPLY. Find the derivative of the function. $$f(x)=csc^{-1}(x^2+1)$$ $$f'(x)=\frac{1}{|x^2+1|\sqrt{(x^2+1)^2-1}} \times 2x$$ $$f'(x)=\frac{2x}{|x^2+1|\sqrt{x^4+1}}$$
      Derivatives involving hyperbolic trigonometry
      Finally, here is a list of the derivatives involving hyperbolic trigonometry.
      $$y=sinhx \\ y'=coshx$$
      $$y=coshx \\ y'=sinhx$$
      $$y=tanhx \\ y'=sech^2x$$
      $$y=cschx \\ y'=-cschxcothx$$
      $$y=sechx \\ y'=-sechxtanhx$$
      $$y=cothx \\ y'=-csc^2x$$
      APPLY. Find the derivative of the function. $$f(x)=xsinhx-coshx$$ $$f'(x)=(x)(sinhx)'+(sinhx)(x)'-(coshx)'$$ $$f'(x)=(x)(coshx)+(sinhx)(1)-(sinhx)'$$ $$f'(x)=xcoshx+sinhx-sinhx$$ $$f'(x)=xcoshx$$
    • derivatives: logs and exponentials
      Derivatives involving logarithms
      We can derive logarithms with either base $10$ or $e$:
      $$y=\log_{b}{x} \\ y'=\frac{1}{xlnb}$$
      $$y=lnx \\ y'=\frac{1}{x}$$
      APPLY. Find the derivative of the function. $$f(x)=ln(sinx)$$ $$f'(x)=\frac{1}{sinx} \times cosx$$ $$f'(x)=\frac{cosx}{sinx}$$ $$f'(x)=cotx$$
      Derivatives involving exponentials
      We can also derive exponentials with base $number$ or $e$:
      $$y=e^x \\ y'=e^x$$
      $$y=b^x \\ y'=b^xlnb$$
      APPLY. Find the derivative of the function. $$f(x)=2e^{2x}$$ $$f'(x)=2e^2x \times 2$$ $$f'(x)=4e^{2x}$$

    • derivatives: application
      Finding the equation for the slope tangent to the curve
      Using the derivative of a function, we can find the slope tangent to a point on the curve. To do this, take the derivative and find a point on the graph. Plug in those values to the point slope formula accordingly, $y-y_1=m(x-x_1)$ where the derivative is $m$ and the points are $(x,y)$.
      Finding critical points
      In addition to finding the points tangent to the curve, we can find the critical points of a function. Critical points (see figure) refers to the areas in which the function increases and decreases and the points of the relative minima and maxima (extrema).


      In order to find the critical points of a function, follow these steps:
      1. 1. Find the first derivative of the function.
      2. 2. Find the points where the function is equal to zero or undefined. These points will tell you the exact relative minimum and maximum points.
      3. 3. Take those points and plot on a number line.
      4. 4. With the derivative, find whether the points between the area where the function is equal to zero or undefined have a positive or negative slope by plugging the values into the first derivative; this will help you find the relative minimum and maximum.
      5. 5. Find the second derivative of the function
      6. 6. Find the points where the function is equal to zero or undefined. This will tell you the inflection points.
      7. 7. Take those points and plot on a number line.
      8. 8. With the derivative, find whether the points between the area where the function is equal to zero or undefined have a positive or negative slope by plugging the values into the second derivative; this will tell you whether that area is a concavity up or down.
      APPLY. Find the critical points of the function. $$f(x)=x^2+x-3$$
      $$f'(x)=2x+1$$
      $$2x+1=0$$ $$2x=-1$$ $$x=-\frac{1}{2}$$
      Optimization
      Optimization is the process in which you try to make a material, volume, area, etc. as effective as possible. Conceptually, it's the process to find the minimum, maximum, relative minima and maxima, concavities, and inflection points. You can use this process to optimize:
      1. 1. Read the problem and understand it. Define what is given to you.
      2. 2. Sketch the problem. Label everything as humanly possible.
      3. 3. Write the formula to optimize.
      4. 4. Use constraints to eliminate one variable (the variable you solve for and plug in).
      5. 5. Set the physical domain.
      6. 5. Take the derivative
      7. 6. Find the critical points of the derivative.
      8. 7. Test the critical points and the domain endpoints in the original function to find the minimum and maximum.
      9. 8. Answer the question.

      Not every problem is the same. Here are some scenarios for optimization:
      1. 1. Velocity
      2. 2. Boxes
      3. 3. Triangles
      4. 4. Cost
      5. 5. Viewing angle
      6. 5. Cones
      Related Rates
      With related rates, you can find other related rates.
      1. 1. Read the problem and understand it. Define what is given to you.
      2. 2. Sketch the problem. Label everything as humanly possible.
      3. 3. Relate rates into one equation based on information given.
      4. 4. Take the derivative with respect to time of the equation and chain rule non-constant values.
      5. 5. Substitute the given values and solve for th emissing rate.

      L'Hospitals Rule
      Under L'Hospital's Rule, you can take the limit by simply taking the derivative of the top and bottom (not to be confused with the quotient rule): $$\lim_{x \rightarrow c} \frac{f'(x)}{g'(x)}$$
      Position, Velocity, Acceleration
      By taking the derivative of position $s$, you get velocity $v$ and when you take the derivative of velocity you get acceleration $a$. To find information about each:
      1. 1. Take the first derivative. This will give you the function of velocity.
      2. 2. Set the derivative equal to zero / undefined and find the points.
      3. 3. Plot the numbers on a number line and find whether the area in between the points are positive or negative.
      4. 4. Take the second derivative. This will give you the function of acceleration.
      5. 5. Set the derivative equal to zero / undefined and find the points.
      6. 6. Plot the numbers on a number line and find whether the area in between the points are positive or negative.

      When analyzing the information, we can deduce the following information if we define the given function: $s(t)=at^2+bt+c$ on interval $[x,y]$:
      1. 1. Object is at rest: $t=0$
      2. 2. Object is moving right: $v>0$
      3. 3. Object is moving left: $v<0$
      4. 4. Object changes direction: $v$ changes sign
      5. 5. Object is speeding up: $v$ and $a$ have the same sign
      6. 6. Object is slowing down: $v$ and $a$ have a different sign
      7. 7. Object's total distance: $\mid s(y) - s(x) \mid$
      8. 8. Object's average velocity: $\frac{s(y)-s(x)}{y-x}$
      Mean Value Theorem
      The mean value theorem finds point at which the secant line and the tangent line are parallel. First, you must establish:
      1. 1. $f(x)$ is continuous on interval $[a,b]$
      2. 2. $f(x)$ is differentiable on interval $[a,b]$
      Thus if both rules are true, at least one value $x=c$ where $a<c<b$ such that $f'(c)=\frac{f(b)-f(a)}{b-c}$.
      To find the of $f(x)$ on interval $[a,b]:
      1. 1. Take the derivative.
      2. 2. Use the formula $\frac{f(b)-f(a)}{b-a}$
      3. 3. Set the derivative equal to the value found above.
      4. 3. Make sure the value(s) are in the interval.
