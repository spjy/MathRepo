    • integrals: rules
      Integrals
      An integral finds the area under a curve. The notation of an integral is as follows: $$\int_{a}^{b} f(x)dx$$ where $b$ is the upper limit, $a$ is the lower limit, $f(x)$ is the function, and $dx$ (or $\triangle x$) is displacement of each rectangle.
      Approximation methods
      To approximate the area under the curve, we can use rectangles and take their area. These methods are called left, right, or middle rectangle approximation methods, also known as LRAM, RRAM, and MRAM respectively.

      Let's say we wanted to find the area of the function $x$ with the domain of $[a,b]$ with $n$ rectangles. We simply would employ these steps:
      1. 1. Choose the domain of where you want to approximate the area and how many rectangles you want to use. The more rectangles means that your answer is more accurate.
      2. 2. Find the width of each rectangle of the function by using the equation $w=\frac{b-a}{n}$.
      3. 3. Create a table of values and increase each value by your width while staying within the domain.
      4. 4. For LRAM approximation, multiply each $y$ value by the width of each rectangle except for the last. For RRAM approximation, multiply each $y$ value by the width of each rectangle except for the first. Finally, for MRAM approximation, take the average of each $x$ value, make a table of values based on your original function and multiply each $y$ value by the height.
      Numerical Methods (Trapezoidal Approximation)
      If an integral is impossible to compute, we can use numerical methods.
      1. $$A=(\frac{y_2+y_1}{2}) \triangle x$$
      Limit definition
      To find the integral of $\int_{a}^{b} f(x)dx$ with the limit definition of an integral, use the steps:
      1. 1. Find $\triangle x$ and $x$: $$\triangle x=\frac{b-a}{n}$$ $$x=a+(k)(\triangle x)$$
      2. 2. Plug in $x$ for each $x$ of the function and plug in $\triangle x$ for $dx$.
      3. 3. Take the limit as $n \rightarrow \infty$ and sum each k.
      The Fundamental Theorem of Calculus (FTC)
      The holy grail of calculus! You now have the key to all power! Here is the key: $$\int_{a}^{b} f'(x)dx=f(b)-f(a)$$ on the interval $[a,b]$. This rule basically states that the integral is the antiderivative.

      Note: $f(b)-f(a)$ is written as $f(x) |^b_a$.
      Indefinite Integral
      An indefinite integral is a general equation that has no limits. To denote this difference, we must add a $+c$ after integrating.
      Definite Integral
      A definite integral is one that has limits and is solvable.
      Power Rule
      $$\int x^n=\frac{x^{n+1}}{n+1}+c$$
      Substitution (Reverse Chain Rule)
      We can find the integral using substitution, or the reverse chain rule. If we define the substitution variable as u, follow these steps:
      1. 1. Strategically choose the term in the function you want to substitute.
      2. 2. Define it as $u$.
      3. 3. Take the derivative of the term, solve for $dx$ in terms of $u$.
      4. 4. Change the limits in terms of $u$. Plug in the current limits in the substitution equation for $x$.
      5. 5. Integrate the function in terms of $u$.
      Parts (Reverse Product Rule)
      We can find the integral using parts, or the reverse product rule. This method basically "transforms" the current integral into another form so we can take the integral easily. A good candidate for this method is if you need to integrate the product of two dissimilar functions, for instance, $\displaystyle \int (polynomial)(exponential) dx$.
      1. 1. Separate the two dissimilar functions into parts, one equal to $u$ and the other function and everything else equal to $dv$.
      2. 2. The general rule of thumb for what to define $u$ is LIPET; however, be aware that this does not always work.
        1. a. Logarithms
        2. b. Inverse Trigonometry
        3. c. Polynomials
        4. d. Exponentials
        5. e. Trigonometry
      3. 3. For the function defined by $u$, take the derivative of $u$ in terms of the function variable and solve for $du$.
      4. 4. For the function defined by $v$, integrate it.
      5. 5. After deriving $u$ and integrating $v$, plug in to this equation: $$uv-\int vdu$$
      Partial Fraction Decomposition
      If a function is too difficult to be integrated, we can simplify it through partial fraction decomposition. If we have the integral in the form $\displaystyle \frac{N(x)}{D(x)}$:
      1. 1. If $N(x)$ is larger than $D(x)$, long divide it.
      2. 2. Factor $D(x)$
      3. 3. Consider these cases:
      4. a. $\displaystyle (ax+b) => \frac{A}{ax+b}$
      5. b. $\displaystyle (ax+b)^n => \frac{A}{ax+b} + \frac{B}{(ax+b)^2} + \frac{C}{(ax+b)^3} + \frac{N}{(ax+b)^n}$
      6. c. $\displaystyle (ax+b)^n => \frac{Ax+B}{ax^2+bx+c}$
      7. d. $\displaystyle (ax^2+bx+c)^n => \frac{Ax+B}{ax^2+bx+c} + \frac{Cx+D}{(ax^2+bx+c)^2} + \frac{Ex+F}{(ax^2+bx+c)^3} + \frac{Nx+(N+1)}{(ax^2+bx+c)^n}$
      8. 4. Clear fractions by multiplying $D(x)$.
      9. 5. Solve the values as a system of equations.
      Trigonometric Substitution
      For integrals that look similar to the inverse trigonometric integrals, we can use trig subtitution:
      1. 1. Identify the type of substitution to make:
      2. a. $const+x^2=tan^2(\theta)$
      3. b. $const-x^2=sin^2(\theta)$
      4. c. $x^2-const=sec^2(\theta)$
      5. 2. Rearrange the $x^2$ and $const$ terms with trigonometric identities.
      6. 3. Substitute the $x^2$ term with the the respective type, as shown above.
      7. 4. Differentiate the substitution and solve for $dx$.
      8. 5. Plug in each term you solve for.
      9. 6. Integrate normally.
      10. 7. Solve for $\theta$ and create a right triangle.
      11. 8. Using the triangle, convert the integral back om terms of $x$.
      Weierstrass Substitution ($\displaystyle tan(\frac{x}{2})$ substitution)
      If we encounter an integral that is not able to be integrated with the previous methods, we can use Weierstrass substitution.
      1. 1. Substitute $\displaystyle u=\frac{x}{2})$
      2. a. $\displaystyle sinx=\frac{2u}{1+u^2}$
      3. b. $\displaystyle cosx=\frac{1-u^2}{1+u^2}$
      4. c. $\displaystyle dx=\frac{2du}{1+u^2}$
      5. 2. Substitute each term and integrate.
      6. 3. Substitute back in for $u$.
      Improper Integrals
      If an integral contains an infinity or an asymptote, try these methods:
      1. 1. Integrals involving infinity: $\displaystyle \int_{0}^{\infty} ax^2 dx => \lim_{b \rightarrow \infty} \int_{0}^{b} ax^2 dx$
      2. a. Integrals involving vertical asyptotes: $\displaystyle \int_{-1}^{1} \frac{1}{x} dx => \lim_{b \rightarrow 0} \int_{-1}^{0} \frac{1}{x} dx + \int_{0}^{1} \frac{1}{x} dx$

      Integrals involving trigonometry
      $$\int sinx \ dx \\ =-cosx+c$$
      $$\int cosx \ dx \\ =sinx+c$$
      $$\int tanx \ dx \\ =-ln|cosx|+c$$
      $$\int cscx \ dx \\ =ln|cscx-cotx|+c$$
      $$\int secx \ dx \\ =-ln|secx-tanx|+c$$
      $$\int cotx \ dx \\ =ln|sinx|+c$$
      Integrals involving inverse trigonometry
      $$\int \frac{1}{\sqrt{1-x^2}} \ dx \\ =sin^{-1}x+c$$
      $$\int \frac{1}{1+x^2} \ dx \\ =tan^{-1}x+c$$
      $$\int \frac{1}{x\sqrt{x^2-1}} \ dx \\ =sec^{-1}|x|+c$$
    • integrals: application
      Planar areas
      We can find the area between two functions by integration.

      If we are integrating the functions in terms of $x$, $y_2$ and $y_1$, where they intersect at points $b$ and $a$, follow these steps:
      1. 1. Find when each function intersects each other by making them equal to each other. These values are in terms of the $x$ axis and are the upper and lower bounds.
      2. 2. Graph the function to see which function is on top or bottom.
      3. 3. We subtract the functions, $top_{function}-bottom_{function}$.
      4. 4. Set up the integral: $\int_{a}^{b} top_{function}-bottom_{function} \ dx$
      5. 5. Solve.

      We can also find the planar area of a function in terms of $y$.
      1. 1. Find when each function intersects each other by making them equal to each other. These values are in terms of the $y$ axis and are the upper and lower bounds.
      2. 2. Graph the function to see which function is on right or left.
      3. 3. We subtract the functions, $right_{function}-left_{function}$.
      4. 4. Set up the integral: $\displaystyle \int_{a}^{b} right_{function}-left_{function} \ dx$
      5. 5. Solve.
      Volume of solids of revolution using washers
      If we rotate a function around an axis, a three dimensional shape can be created. We must first consider the bounds, the function, and the axis of rotation. The equation for the volume of washer is $$V= \pi r^2 \ \triangle x$$
      1. 1. To find the bounds of the shape, simply find the intersection points of the functions. The limits will be in terms of the axis of rotation.
      2. 2. The general equations are: $$top^2-bottom^2$$ $$(axis-outer \ radius)^2-(axis-inner \ radius)^2$$
      3. 3. We must find the infinite sum of the shape to find the the full volume. We take the limit as $x$ approaches infinity, then set up the integral as $a$ approaches $b$: $$V= \pi \lim_{x \rightarrow \infty} \sum (axis-outer \ radius)^2-(axis-inner \ radius)^2 \triangle x$$ $$V= \pi \int_{a}^{b} (axis-outer \ radius)^2-(axis-inner \ radius)^2 \ dx$$
      4. 4. Solve.
      5. Note: If the axis is above/right, subtract $axis-function$ but if it's below/left, add $axis+function$.
      Volume of solids of revolution using shells
      Another method to find the volume of a shape rotated around an axis is the shell method. Like the washer method, we must consider the bounds, the function, and the axis rotation. The equation for the volume of a shell is: $$V=2 \pi rh \triangle x$$
      1. 1. To find the bounds of the shape, simply find the intersection points of the functions. The limits will be in terms of the axis perpendicular to the one of rotation.
      2. 2. The general equation is: $$(x-axis)-(top_{function}-bottom_{function})$$
      3. 3. We must find the infinite sum of the shape to find the the full volume. We take the limit as $x$ approaches infinity, then set up the integral as $a$ approaches $b$: $$V= 2 \pi \lim_{x \rightarrow \infty} \sum (x-axis)-(top_{function}-bottom_{function}) \triangle x$$ $$V= 2 \pi \int_{a}^{b} (x-axis)-(top_{function}-bottom_{function}) \ dx$$
      4. 4. Solve.
      5. Note: If the axis is above/right, use $x-axis$ but if it's below/left, use $x+axis$.
      Volume of solids of with known cross-sections
      If we are given what type of shape the solid is, then we can find the volume. The general equation is: $$V=area \times thickness$$
      1. 1. The base of the cross section is $top-bottom$ and the area is in terms of the $x$ axis.
      2. 2. The geometry depends on the shape of the cross sections.
      3. 3. The integral depends on the geometry of the base and cross section.
      4. 4. Solve.

      Arc length
      We can find the arc length by using: $$ dL= \sqrt{1+(\frac{dy}{dx})^2} dx$$ If it is in terms of y, use $$ \displaystyle dL= \sqrt{1+(\frac{dy}{dx})^2}dy$$
      Surface Area
      We can find the surface area by using: $$SA=2 \pi r dL$$ where dL is the arc length.
      Position, Velocity, Acceleration
      We can find find the position, velocity, and acceleration of a function by integrating.
      1. 1. The integral of acceleration is velocity.
      2. 2. The integral of velocity is displacement.
